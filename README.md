# Task Platform (Go + PostgreSQL + JWT)

A full-stack task management web application built with **Go (Golang)**, **PostgreSQL**, and **JWT authentication**, using server-side rendered templates (HTML + Tailwind CSS).

This project demonstrates backend development skills including authentication, middleware, database integration, and clean architecture.

---

##  Features

- User registration & login
- JWT authentication (stored in HTTP-only cookies)
- Protected routes using middleware
- Create / View / Delete tasks
- User-specific task management
- Server-side rendered UI (Go templates)
- Modern UI using Tailwind CSS
- PostgreSQL database integration

---

## Tech Stack

- **Backend:** Go (net/http)
- **Database:** PostgreSQL
- **Auth:** JWT (github.com/golang-jwt/jwt)
- **Password Hashing:** bcrypt
- **Frontend:** HTML templates + Tailwind CSS
- **Architecture:** MVC-style modular structure
